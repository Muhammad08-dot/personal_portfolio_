### 6. `vite-plugin-singlefile` vs React Router Conflict
`vite-plugin-singlefile` bundles everything into a single `index.html` with all assets inlined. This conflicts with client-side routing since deep-link URLs (e.g., `/projects`) may not resolve correctly without a server-side redirect.

**The `vercel.json` file handles this partially**, but testing is recommended — especially navigating directly to any non-root route after deployment.
