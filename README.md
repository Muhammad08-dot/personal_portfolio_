# 💻 Muhammad Abdullah - Personal Portfolio

A highly immersive, cyber-terminal themed personal portfolio showcasing my experience as an AI/ML & Agentic AI Engineer. The project leverages a dark, matrix-inspired aesthetic paired with high-performance 3D components and smooth animations to deliver a premium, fully responsive user experience.

## ✨ Features
- **Cyber-Terminal Aesthetic**: Monospaced typography, neon-green matrix highlights, and custom console window frames.
- **Interactive 3D Environments**: Built-in HTML5 Canvas and WebGL scenes (via Three.js) for floating geometry and interactive backgrounds.
- **Command Palette (`⌘K`)**: Quick navigation menu replicating IDE-like workflows.
- **Responsive Mobile-First Design**: Fluid grid layouts that adapt flawlessly from large desktop monitors down to mobile touch screens.
- **Dynamic Routing & Animations**: Seamless page transitions and micro-interactions powered by Framer Motion and React Router.

---

## 🛠️ Tech Stack & Frameworks

This portfolio is engineered using modern, lightning-fast web technologies:

### Core Architecture
* **[React 19](https://react.dev/)**: The core UI library used for building interactive, component-based architectures.
* **[TypeScript](https://www.typescriptlang.org/)**: Provides strict static typing across all components and configuration objects, ensuring code safety and scalability.
* **[Vite 7](https://vitejs.dev/)**: Next-generation frontend tooling providing lightning-fast Hot Module Replacement (HMR) and optimized production builds.

### Styling & UI
* **[Tailwind CSS v4](https://tailwindcss.com/)**: A utility-first CSS framework used for highly customized, responsive, and mobile-first layouts without leaving the JSX.
* **[Lucide React](https://lucide.dev/)**: A clean, consistent icon library utilized for navigational and structural UI iconography.
* **Custom CSS Variables**: Extensively used to maintain the strict `#0A0A0A` background and `#00FF00` matrix-green color scheme.

### 3D Graphics & Animations
* **[Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)**: Powers the hardware-accelerated 3D scenes (like the Hero floating geometry) directly within the React ecosystem.
* **[Framer Motion](https://www.framer.com/motion/)**: Handles the complex, declarative animations, scroll-triggered fade-ups, and fluid page transitions.

### Utilities
* **[React Router DOM](https://reactrouter.com/)**: Manages client-side routing across the About, Skills, Experience, Projects, and Connect pages.
* **clsx & tailwind-merge**: Utilities for constructing dynamic class names and safely merging Tailwind classes.

## 🚀 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Muhammad08-dot/personal_portfolio_.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
